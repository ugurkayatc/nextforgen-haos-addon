# NextForGen HAOS Add-on

Official Home Assistant OS add-on for the NextForGen Smart Home Platform.

## Installation

Settings → Add-ons → Add-on Store → menu → Repositories, paste:

    https://github.com/ugurkayatc/nextforgen-haos-addon

Install **NextForGen Agent**, enter the bootstrap code from your technician panel, start.
